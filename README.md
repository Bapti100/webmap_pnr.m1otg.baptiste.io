## Overview

**PNR Bauges** is an interactive web platform dedicated to the **Bauges Massif Regional Natural Park**, located in the French Prealps. This site provides a comprehensive overview of the park’s geography, cultural and natural heritage, biodiversity, and territorial planning. It features interactive maps, key figures, and thematic content to raise awareness and support sustainable management of the territory.

---

## Features

### 🗺️ **Interactive Maps**
- Built with **ArcGIS Embedded Maps** for seamless integration of geographic data.
- **Thematic layers**: Protected areas, biodiversity indicators, hiking trails, and cultural heritage sites.
- **Dynamic exploration**: Users can interact with maps to discover specific locations, ecosystems, and projects.

### 📊 **Key Figures & Data Tables**
- **Geographic data**: Area, population, elevation, and administrative divisions.
- **Biodiversity metrics**: Number of species, protected habitats, and conservation projects.
- **Land use statistics**: Vegetation, urban, and activity zones.

### 🏞️ **Thematic Sections**
- **Territorial Observatory**: Centralizes data for sustainable and collaborative management.
- **Geography**: Key geographic features, elevation, and landscapes.
- **Cultural Heritage**: Traditional architecture, local crafts, and events.
- **Natural Heritage**: Flora, fauna, and conservation actions.
- **Territorial Planning**: Urban/rural balance, climate adaptation, and local projects.

### 📱 **Responsive Design**
- Adaptive layout for desktop, tablet, and mobile devices.
- Optimized for accessibility and user experience.
